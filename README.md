
Created for LeanHackathon by [Emergence AI](https://emergence.ai)

Following are the files:

1. **PicTreeLang**: Inspired by [Cope and Drag (Brown CS)](https://cs.brown.edu/people/sk/Publications/Papers/Published/pgnk-cope-and-drag/). Essential idea is to formally verify visualisations of tree structures. We do it with the restricted case of MaxHeaps by focussing on a few "nice" properties.

2. **CompLang**: This is essentially a compiler for a small programming language without loops, but allowing variable mutations.

3. **MonadicCompLang**: We repeat the exercise in CompLang, but this time using Monads.
